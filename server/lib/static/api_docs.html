üçï Dishes API

GET /api/dishes
Get all dishes
Request
curl http://localhost:8080/api/dishes
Response
{ "success": true, "data": [ { "id": 1, "name": "Pizza Margherita", "photoUrl": "https://example.com/pizza.jpg", "price": 12.99 } ] }
GET /api/dishes/{id}
Get a specific dish by ID
id integer Dish ID (path parameter)
Request
curl http://localhost:8080/api/dishes/1
Response
{ "success": true, "data": { "id": 1, "name": "Pizza Margherita", "photoUrl": "https://example.com/pizza.jpg", "price": 12.99 } }
POST /api/dishes
Create a new dish
name string Dish name (required)
photoUrl string Photo URL (required)
price number Price (required)
Request
curl -X POST http://localhost:8080/api/dishes \ -H "Content-Type: application/json" \ -d '{ "name": "Burger", "photoUrl": "https://example.com/burger.jpg", "price": 9.99 }'
PUT /api/dishes/{id}
Update an existing dish
id integer Dish ID (path parameter)
name string Dish name
photoUrl string Photo URL
price number Price
Request
curl -X PUT http://localhost:8080/api/dishes/1 \ -H "Content-Type: application/json" \ -d '{ "name": "Updated Pizza", "photoUrl": "https://example.com/pizza2.jpg", "price": 14.99 }'
DELETE /api/dishes/{id}
Delete a dish
id integer Dish ID (path parameter)
Request
curl -X DELETE http://localhost:8080/api/dishes/1
Response
{ "success": true, "message": "Dish deleted successfully" }

üì¶ Orders API

GET /api/orders
Get all orders
Request
curl http://localhost:8080/api/orders
Response
{ "success": true, "data": [ { "id": 1, "userId": 2, "phone": "0987654321", "dishId": "1", "latitude": 40.7128, "longitude": -74.0060, "address": "123 Main St, New York, NY", "completed": false } ] }
GET /api/orders/{id}
Get a specific order by ID
id integer Order ID (path parameter)
POST /api/orders
Create a new order
userId integer? User ID (optional)
phone string Phone number (required)
dishId string Dish ID (required)
latitude number Latitude (required)
longitude number Longitude (required)
address string Delivery address (required)
completed boolean Completion status (default: false)
Request
curl -X POST http://localhost:8080/api/orders \ -H "Content-Type: application/json" \ -d '{ "userId": 1, "phone": "1234567890", "dishId": "1", "latitude": 40.7128, "longitude": -74.0060, "address": "123 Main St", "completed": false }'
PUT /api/orders/{id}
Update an existing order
DELETE /api/orders/{id}
Delete an order
